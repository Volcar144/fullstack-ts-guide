# Welcome to the TypeScript Full-Stack Guide

Whether you've never written a line of code or you've tinkered with a bit of HTML, this guide will take you all the way from **the absolute basics** to shipping a real full-stack web app.

## What you'll learn

- **The foundations** — HTML structure, JavaScript logic, TypeScript types
- **React** — components, state, hooks, and how UIs are built today
- **Next.js + Tailwind** — building a full app step by step
- **The ecosystem** — Prisma, BetterAuth, Sentry, PostHog

## How to use this guide

Work through the sections in order. Each page builds on the last. Code examples are everywhere, and you can try them live in the [Playground](/playground).

<Callout type="info">
  **New to coding?** Start at [Choosing a Framework](/docs/foundations/choosing-a-framework). If you already know some JS, jump ahead to the React section.
</Callout>

## The Playground

This site includes a **live code playground** where you write React + Tailwind and see the result instantly — no setup required. [Open the Playground →](/playground)
