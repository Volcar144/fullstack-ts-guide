# Tailwind CSS Fundamentals

Tailwind is a **utility-first** CSS framework. Instead of writing CSS files with class names like `.card`, you compose small single-purpose classes directly in your HTML/JSX.

## The core idea

```tsx
// Traditional approach
// HTML
<div class="card">
  <h2 class="card-title">Hello</h2>
</div>

// CSS
.card { background: white; border-radius: 8px; padding: 24px; box-shadow: ... }
.card-title { font-size: 1.5rem; font-weight: bold; margin-bottom: 8px; }

// Tailwind approach — no CSS file needed
<div className="bg-white rounded-xl p-6 shadow">
  <h2 className="text-2xl font-bold mb-2">Hello</h2>
</div>
```

## Essential classes

### Spacing
```
p-4      padding: 1rem (16px)
py-2     padding top + bottom: 0.5rem
px-6     padding left + right: 1.5rem
mt-4     margin-top: 1rem
space-y-4  gap between stacked children
```

### Typography
```
text-sm      font-size: 0.875rem
text-base    font-size: 1rem
text-xl      font-size: 1.25rem
text-2xl     font-size: 1.5rem
font-bold    font-weight: 700
font-medium  font-weight: 500
text-gray-600  color: #4b5563
truncate     overflow hidden + ellipsis
```

### Layout
```
flex           display: flex
flex-col       flex-direction: column
items-center   align-items: center
justify-between  justify-content: space-between
gap-4          gap: 1rem

grid           display: grid
grid-cols-3    grid-template-columns: repeat(3, 1fr)

hidden         display: none
block          display: block
```

### Sizing
```
w-full    width: 100%
w-1/2     width: 50%
w-64      width: 16rem (256px)
h-screen  height: 100vh
min-h-0   min-height: 0
max-w-md  max-width: 28rem
```

### Colors
```
bg-blue-500        background-color
bg-blue-500/50     50% opacity
text-red-600       text color
border-gray-200    border color
ring-2 ring-blue-500  focus ring
```

### Responsive
Prefix any class with a breakpoint:
```tsx
<div className="
  grid 
  grid-cols-1          /* mobile: 1 column */
  md:grid-cols-2       /* md (768px+): 2 columns */
  lg:grid-cols-3       /* lg (1024px+): 3 columns */
">
```

### State variants
```tsx
<button className="
  bg-blue-500
  hover:bg-blue-600    /* on hover */
  focus:ring-2         /* on focus */
  active:scale-95      /* while clicking */
  disabled:opacity-50  /* when disabled */
  transition-colors    /* smooth color transitions */
">
```

## Dark mode
```tsx
<div className="bg-white dark:bg-gray-900 text-gray-900 dark:text-white">
```

## The `cn` utility

When you need to conditionally apply classes, use `clsx` + `tailwind-merge`:

```tsx
import { cn } from '@/lib/utils';

function Button({ variant, className, ...props }) {
  return (
    <button
      className={cn(
        'px-4 py-2 rounded font-semibold',
        variant === 'primary' && 'bg-blue-500 text-white',
        variant === 'ghost' && 'bg-transparent hover:bg-gray-100',
        className, // allow external classes to override
      )}
      {...props}
    />
  );
}
```
